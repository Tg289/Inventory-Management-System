# Inventory Management System

## Description
A simple desktop-based Inventory Management System in Java with MySQL.  
Allows adding, updating, deleting, and viewing items with a GUI interface.

## Features
- Add new items
- Update existing items
- Delete items
- List all items

## Tech Stack
- Java (Swing for GUI)
- MySQL
- JDBC
